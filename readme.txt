== Description ==

Convert all images into lightbox except for those images that are part of the exceptions.

Features

* Supports cloudinary
* Automatic lightbox with exception

Read documentation <a href="https://ziorweb.dev/docs/zior-lightbox.html" target="_blank">here</a>

Initial release.

== Frequently Asked Questions ==

= Default settings =

When activated, all images will be converted into light box except for those excempted in the settings.
To add exceptions, go to Settings -> ZIOR Lightbox
